LAMA documentation

LAMA (Lightweight Analysis of Morphological Abnormalities) is an open source set of tools to automatically identify abnormal morphology.
It was originally developed for use with whole-embryo ct scans of knockout mice, but has been shown to work with other data types.

These docs are a work in progress, and will be updated shortly.

If you have any questions or comments, please contact us at sig@har.mrc.ac.uk
