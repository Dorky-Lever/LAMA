Getting started
 

Created Friday 22 January 2016

Installation



Requirements:

  • A windows or Linux PC with Python 2.7 installed. It should work on Mac. but has not been tried yet
  • elastix (http://elastix.isi.uu.nl/|) - tested with version 4.7
  • The R statstical package (https://www.r-project.org/) - tested with version 3.1.2


Linux users

  • Download elastix here http://elastix.isi.uu.nl/download_links.php and install
  • Or (On debian/ubuntu) use apt-get install elastix

  • To install some dependencies run setup.py in the root of the LAMA directory

  • SimpleITK can sometimes be problematic to install. If you get import error complaing about SimpleITK, try the following
    • At a terminal try the following, which installs SimpleITK with the home directory

$ wget http://sourceforge.net/projects/simpleitk/files/SimpleITK/0.9.0/Python/SimpleITK-0.9.0-py2.7-linux-x86_64.egg 
$ easy_install --user SimpleITK-0.9.0-py2.7-linux-x86_64.egg

  • If there are Import Error messgaes complainig about psutil, You may need to install the python development package for psutil to work
    • Ubuntu: sudo apt-get install python-dev
    • fedora: sudo dnf install python-devel
    • After installation, run setup_LAMA.py again


Windows users

  • Download elastix here http://elastix.isi.uu.nl/download_links.php and install
  • Install Python 2.7. Good options are Anaconda ( https://www.continuum.io/downloads) and WinPython (http://winpython.github.io/) which includes scientific packages required by LAMA.
  • Download the LAMA source code from here: our github repo
  • Unpack
  • There are some executable scripts in lama/windows to make running lama easier
  • Run windows/setup_LAMA which will download and install a few required Python libraries, and install them locally.
  • lama.py, pheno_detect.py, and lama_stats.py can all be run by running the respective files in this folder
